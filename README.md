# Proyecto: Impacto del Mal Uso de Dispositivos en Niños

Este proyecto consiste en una página informativa que explica los riesgos asociados al uso excesivo de teléfonos, tablets y computadores en niños, así como recomendaciones para su uso saludable.  

La finalidad es aumentar la conciencia sobre los peligros físicos, cognitivos y socioemocionales del uso prolongado de pantallas durante la infancia.

---

##  Propósito del Proyecto

- Sensibilizar a padres, docentes y cuidadores sobre los efectos del uso inadecuado de dispositivos electrónicos.
- Presentar información clara, accesible y basada en principios de diseño web moderno.
- Ofrecer recursos y un formulario para solicitar más información sobre prácticas saludables en el uso de tecnología infantil.

---

##  Semántica HTML Aplicada

El código utiliza etiquetas semánticas HTML5 para mejorar la estructura y la comprensión del contenido:

- `<header>`: Contiene el título y descripción general.
- `<nav>`: Navegación del sitio con enlaces internos.
- `<main>`: Contenido principal de la página.
- `<section>`: Agrupa áreas temáticas como riesgos, recomendaciones y recursos.
- `<article>`: Se usa para contenidos individuales dentro de secciones.
- `<aside>`: Presenta información complementaria.
- `<footer>`: Información final del sitio.

Este uso semántico mejora:
- SEO
- Comprensión por lectores de pantalla
- Mantenibilidad del código

---

##  Accesibilidad Implementada

Para garantizar un contenido accesible:

- Se agregó `lang="es"` al elemento `<html>`.
- Los enlaces de la barra de navegación incluyen `aria-label` para accesibilidad.
- Todas las imágenes cuentan con texto alternativo (`alt`).
- Formularios con etiquetas `<label>` correctamente asociadas a los campos.
- Botones accesibles con `aria-label`.
- Colores suficientemente contrastados.
- Estructura navegable mediante teclado.
